# 2D Link — 技术支持政策

[产品主页](../README_zh-CN.md) · [快速入门](QUICK_START_zh-CN.md) · [用户指南](USER_GUIDE_zh-CN.md) · [兼容性](COMPATIBILITY_AND_PURCHASE_CHECKLIST_zh-CN.md)

本政策适用于 2D Link 1.6.3 Windows x64 正式发行包。

产品支持由实际购买平台处理。Superhive 用户请使用 Superhive 站内消息；通过其他官方渠道购买的用户，请使用对应平台提供的支持联系方式。

## 首次回复目标

在文档化支持边界内，合理产品问题的首次回复目标为 **3 个自然日**。

周末、节假日、销售平台消息投递、问题复杂度和缺失的复现信息可能影响回复时间。数据安全问题和核心 Blender–Photoshop 工作流中可复现的故障优先级最高。

## 支持范围

技术支持覆盖：

- 安装未修改的官方 ZIP；
- 初次 Photoshop 路径配置和 **Test PS** 结果解释；
- 文档化的项目创建、关联、切换和验证；
- 单贴图快速编辑、Reload Textures 和 Auto Reload；
- 关联 PSD/PSB 与 Merge Layers 写回；
- Undo Last Merge 和文档化恢复操作；
- 受支持的静态 JSON 与便捷工具工作流；
- Operation Report 解释；
- 文档化核心流程中可复现的故障。

精确实测宿主版本和兼容候选规则以[兼容性与购买检查清单](COMPATIBILITY_AND_PURCHASE_CHECKLIST_zh-CN.md)为准。

未列出的稳定版本仍可进行初步支持评估。需要隔离版本差异时，可能会要求与已列出的版本进行对比。

## 不包含的服务

技术支持不包含：

- Blender 或 Photoshop 基础教学；
- 长期提供自定义项目修复或私人资产调试服务；
- 自定义管线开发或功能实现；
- 文档工作流之外的第三方插件冲突；
- 修改过的安装包、不完整 ZIP，或由独立包内文件重新拼装的版本；
- macOS、Linux、Beta 或 Nightly 宿主、非 Adobe 编辑器、网络共享、NAS、虚拟文件系统、符号链接、目录联接或延迟同步项目目录的正式支持；
- 超出内置文档化恢复工具的数据恢复；
- 在未来 Blender、Photoshop、Windows、硬件或安全策略进入公开矩阵前保证兼容；
- 法律、税务、支付、退款、收据、下载权限或平台账户协助。

## 发送报告前

请先完成以下检查：

1. 确认安装的是未修改的官方 ZIP。
2. 确认活动项目位于普通本地硬盘。
3. 没有写入操作进行时，重启 Blender 和 Photoshop。
4. Photoshop 路径、执行模式或版本改变后，再次运行 **Test PS**。
5. Auto Reload 没有更新时，先执行一次 **Reload Textures**。
6. 打开 **Open Operation Report**，阅读第一条被阻止的条件和建议操作。
7. Photoshop 仍可能写入时，不要强制清除任务状态或项目锁。
8. 尽可能在项目副本中复现问题。
9. 将报告缩减为仍能复现问题的最小资产和最短操作序列。

## 报告应包含什么

每次消息只提交一个问题，并包含：

- 2D Link 版本；
- 完整 Blender 版本；
- Photoshop 年份和用户可见应用版本；
- Windows 版本；
- 是否安装或运行了多个 Photoshop 版本；
- **Test PS** 显示的 Photoshop 路径和版本；
- 项目存储类型：本地硬盘、网络盘、NAS、云同步目录、符号链接或目录联接；
- 精确复现步骤；
- 预期结果；
- 实际结果和完整可见错误文本；
- 可以提供时，经过检查和脱敏的 'BRANDY_2D_LINK_Last_Report'；
- JSON 问题还应包含导入模式、原始 'skeleton.images' 值、JSON 文件夹结构，以及是否授权外部目录访问。

截图只在能说明界面状态或可见错误时有帮助，不能代替精确复现步骤和错误文本。

## 隐私与共享文件

除非明确要求，不要发送完整生产项目。尽可能使用最小复现项目。

Operation Report 和任务日志可能包含本地路径、账户名、项目名、客户名或未公开美术引用。分享前必须逐项检查并脱敏。

不要包含凭据、支付数据、私人授权信息、机密客户资产或与问题无关的生产文件。

## 恢复与数据安全

使用任何恢复工具前，先查看 **Open Operation Report**。在状态确认前，保留项目文件夹、任务文件夹、项目锁、备份记录和恢复标记。

Photoshop 仍可能写入时，不要手动删除锁或替换项目文件。当前文件不再匹配记录状态时，恢复或撤销可能被拒绝；这是为了防止旧恢复记录覆盖更新后的内容。

内置备份和撤销用于支持文档化工作流，不替代独立备份、版本管理或专业数据恢复。

## 销售平台、更新与未来版本

支付、收据、税务或 VAT、下载权限、退款和平台账户问题由实际购买平台处理。

销售页面和订单条款决定购买包含的支持与更新服务。本仓库不承诺终身支持、终身更新或未来宿主版本兼容性。

新的 Blender 和 Photoshop 公开版本只有在文档化工作流完成测试并更新公开兼容性页面后，才进入支持矩阵。

## 许可证边界

2D Link 正式插件包以 **GPL-3.0-or-later** 分发，并包含对应源码文件和许可证文本。支持与更新权限属于销售平台服务，不改变包内许可证所说明的 GPL 权利。
