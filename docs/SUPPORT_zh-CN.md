# Brandy 2D Link 技术支持政策

本政策适用于 Brandy 2D Link 1.6.3 Windows x64 正式包。

技术支持通过购买所在平台处理。直接销售用户请使用官方包或销售页中提供的支持联系方式。

## 首次回复目标

文档支持范围内的合理产品协助，首次回复目标为 **3 个自然日**。

复杂报告可能需要复现信息或后续确认。回复时间可能受周末、假期、平台消息送达、问题复杂度，以及报告信息完整度影响。

数据安全问题和可复现的 Blender–Photoshop 核心流程故障具有最高优先级。

## 支持范围

技术支持覆盖：

- 未修改官方包的安装；
- 初次 Photoshop 路径配置；
- 文档中的项目关联、贴图刷新、自动刷新、Merge Layers、撤销、恢复、静态 JSON 和便捷工具流程；
- 执行报告解读；
- Blender–Photoshop 核心流程中的可复现故障；
- `README_zh-CN.md` 中列出的已完整测试 Blender 和 Photoshop 版本。

## 已完整测试版本

本政策中的“已完整测试版本”是指以下 Windows x64 Blender 精确版本与 Photoshop 公开版本：

- Blender 4.2.21 LTS、4.3.2、4.4.3、4.5.10、5.0.1、5.1.2；
- Adobe Photoshop CC 2017.1.6、2020 21.2.1、2022 23.5.0、2025 26.10.0、2026 27.7.0。

Blender 按完整版本号核对；Photoshop 按年份与用户可见应用版本核对，Adobe 内部构建号不作为支持门槛。多版本 Photoshop 共存时，以**检测 PS**确认的实际响应路径和应用报告版本为准。未列入清单的稳定版本仍可接受技术支持检查；只有在定位版本差异确有需要时，才会请用户在已列版本中进行对照。

## 不包含的服务

技术支持不包括：

- Blender 或 Photoshop 基础教学；
- 定制项目修复或私人资产调试服务；
- 定制管线开发；
- 文档工作流之外的第三方插件冲突处理；
- 修改包或不完整 ZIP 安装的支持；
- macOS、Linux、Beta/Nightly 宿主程序、网络共享、NAS、虚拟文件系统、符号链接、目录联接或延迟云同步项目目录的支持；
- 超出插件文档化恢复工具范围的数据恢复；
- 未来 Blender、Photoshop、Windows、硬件或安全策略变化进入公开支持矩阵前的兼容性检查；
- 法律、税务、支付或市场账号支持。

## 未来版本

销售页面和订单条款定义购买包含的支持和更新周期。本仓库不承诺终身支持或终身兼容。

Blender 新版本或 Photoshop 新公开版本完成文档工作流测试并更新公开矩阵后，才进入正式支持范围；Photoshop 内部构建号不单独划分支持边界。

## 发送报告前

请先完成以下检查：

1. 确认安装的是未修改的官方 ZIP。
2. 确认项目位于普通本地硬盘目录。
3. 没有写入任务正在执行时，可重启 Blender 和 Photoshop。
4. Photoshop 路径、调用方式或版本发生变化后，重新执行**检测 PS**。
5. 自动刷新未更新画面时，先执行一次**手动刷新 / Reload Textures**。
6. 打开**执行报告**并阅读第一条被阻止条件。
7. Photoshop 仍可能写入文件时，不要强制清除任务状态或项目锁。
8. 可以时，在项目副本中复现问题。

## 报告应包含什么

每次报告只提交一个问题，并提供：

- Brandy 2D Link 版本；
- 完整 Blender 版本；
- Photoshop 年份与用户可见应用版本；
- Windows 版本；
- 是否安装或运行了多个 Photoshop 版本；
- **检测 PS**显示的实际 Photoshop 路径与应用报告版本；
- 项目是否位于本地硬盘、网络盘、NAS、云同步目录、符号链接或目录联接；
- 精确复现步骤；
- 预期结果；
- 实际结果和完整可见错误文字；
- 只有在有助于说明问题时才提供截图；
- 可以提供时，经过检查和脱敏的 `BRANDY_2D_LINK_Last_Report`；
- JSON 导入/导出问题需说明导入模式、原始 `skeleton.images` 值、JSON 文件夹结构，以及是否授权外部贴图目录。

发送文件或报告前，请移除私人路径、账户名、客户名、未公开美术、凭据、支付信息和任何机密生产资料。

## 隐私与文件

除非明确要求，不要发送完整生产项目。优先使用最小复现项目。

执行报告即使替换了部分已知路径，仍可能包含本地诊断信息。分享前请自行检查并脱敏。

## 商店与退款问题

支付、收据、税务/VAT、下载权限、退款请求和市场账号问题由实际购买平台处理。请遵循该平台的政策和支持入口。

对于兼容候选环境，请先对照公开支持矩阵和文档化工作流限制进行确认。

## 许可证边界

官方 Brandy 2D Link 插件包以 **GPL-3.0-or-later** 发布。支持与更新访问属于购买平台定义的服务。GPL 权利以插件包随附许可证文本为准。
